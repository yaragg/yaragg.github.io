Esse é um projeto em dupla que fiz para uma matéria de web usando HTML5 e JavaScript. Uma das funcionalidades interessantes de projeto é a detecção de movimento: experimente dar permissão para acessar a webcam e clicar na caixa de 'Motion detection'! Se além disso você habilitar a caixa 'Camera controls boost', os sons graves ficarão mais fortes quanto mais você se movimentar. Acesse online! (Em inglês.)
