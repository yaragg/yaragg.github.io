This was an assignment for my Programming Lab II course at the University of SÃ£o Paulo. For this project, we had to create a simple programming language that can be used to program robots to fight and gather resources. The player can write scripts to define their robots' behavior and once those scripts are loaded and assigned to each robot, a graphical simulation is run and shows the ensuing battle.

The main code was written in Java and used JavaCC to create a simple compiler for our language.
